# OVA Base de Datos

Objeto Virtual de Aprendizaje sobre Búsqueda en Bases de Datos

## Instalación

\`\`\`bash
npm install
\`\`\`

## Desarrollo

\`\`\`bash
npm run dev
\`\`\`

## Construcción

\`\`\`bash
npm run build
\`\`\`

- ✅ API Routes
- ✅ Responsive Design
